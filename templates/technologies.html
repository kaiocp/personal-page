Ferramentas

HTML 5

Conhecida como Linguagem de Marcação de Hipertexto (ou, em inglês, Hypertext Markup Language), é uma ferramenta utilizada para construir páginas web que serão interpretadas por navegadores. Seu site oficial é o https://www.w3.org/html/

CSS 3

Cascading Style Sheets ou Folha de Estilo em Cascatas é um recurso utilizado para estilizar um documento web. É a ferramenta que dá estilo ao HTML. Site oficial: https://www.w3.org/Style/CSS/

JavaScript

De acordo com a página oficial da Wikipédia, JavaScript "é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma (protótipos, orientado a objeto, imperativo e, funcional)." No contexto do desenvolvimento web, o JS é responsável por definir o comportamento de um website. Site oficial: https://www.javascript.com/

Node.js

O Node.js é um software que permite a execução d códigos JavaScript fora de navegadores web, uma vez que ela só pode ser interpretada dentro destes. Isto permite que o JS, uma linguagem usada principalmente no front-end, seja usada também no back-end. O site oficial é https://nodejs.org/
